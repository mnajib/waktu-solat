# waktu-solat
Waktu solat JAKIM di terminal

# Requirement
bash, xmlstarlet

# Penggunaan
``` bash
waktu-solat
waktu-solat SGR04
waktu-solat "Kuala Lipis"
```
Rujuk [JAKIM](http://www2.e-solat.gov.my/zon-waktusolat.php) untuk mengetahui kod kawasan.
